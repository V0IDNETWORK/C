# 🇬🇧 English

## 📖 About This Repository

Welcome to my **C Programming Course Repository**.

This repository is part of my journey to deeply understand the **C programming language**, computer systems, memory management, and low-level programming concepts.

Here I document my exercises, experiments, and projects while learning how software interacts with hardware and operating systems.

---

## 🧠 Learning Goals

The main goals of this repository:

- Understand how programs work internally
- Learn memory management
- Master pointers and low-level concepts
- Build system programming foundations
- Improve problem-solving skills
- Create practical C projects

# 🇮🇷 فارسی

## 📌 درباره این مخزن

به ریپازیتوری دوره برنامه‌نویسی زبان **C** خوش آمدید.

این مخزن بخشی از مسیر یادگیری من برای درک عمیق زبان C، سیستم‌های کامپیوتری، مدیریت حافظه و برنامه‌نویسی سطح پایین است.

در اینجا تمرین‌ها، آزمایش‌ها و پروژه‌هایی که در طول یادگیری انجام می‌دهم ثبت می‌شوند.
